# RentalCar Product Blueprint

## Product Vision
RentalCar is an AI-powered vehicle rental and car listing platform.
The platform combines rental flows, car sale listings, and AI-assisted decision support features in a single system.

## Core Modules
- Public Marketplace
- Rentals
- Sales / Listings
- Users
- AI Assistant
- Pricing
- Trust / Reliability
- Favorites
- Admin / Moderation

## User Roles
- Visitor
- Registered User
- Seller / Listing Owner
- Renter
- Admin

## MVP Scope
- Authentication
- Car listing
- Car detail page
- Rental flow
- Sale listing creation
- Favorites
- Basic filtering
- AI assistant (basic)
- Price analysis (basic)
- Trust analysis (basic)
- Basic admin listing management

## Final Vision
- Natural language filtering
- Personalized recommendations
- Listing quality score
- User trust profile
- Moderation workflow
- Buy vs Rent advisor
- AI-assisted listing creation
- Advanced admin dashboard
